## win + 系列相关快捷键

```shell
win				# 打开"开始"菜单
win + v			# 打开剪切板
win + d			# 回到桌面
win + E			# 打开文件资源管理器
win	+ i			# 打开设置
win	+ N			# 打开“通知”面板
win + G			# 打开"xbox"程序
win + R			# 打开"运行"对话框
win	+ Tab		# 3D窗口切换
win + Q			# 打开"开始"菜单的搜索框
win + T			# 查看"任务栏"信息
win + U			# 打开"辅助功能设置”菜单
win + P			# 切换显示屏模式
win + L			# 锁定界面
win	+ A			# 打开"蓝牙、WLAN"相关设置对话框
win	+ H			# 打开"语言服务"对话框
win + K			# 打开"投放"
win	+ X			# 调用"关机"相关设置菜单
win + M			# 切换到桌面
win + +|-   	# 调用放大镜进行缩放
win + Pause		# 调用系统信息
win + S			# 打开"开始"菜单的搜索框
```

## 显示屏

```sh
# 切换应用程序所在显示器
win + shift + '箭头'
```
