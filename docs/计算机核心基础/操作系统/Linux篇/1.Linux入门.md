### 命令语法

> 有多个命令选项时，除第一个外其他都可以不用加 **-**

```shell
command [-option] [paramenter]

命令 + 可选的行为细节 + 参数
```

## 服务管理

- firewalld：防火墙
- mysqld：mysql服务器

```sh
# 查看服务运行状态
systemctl status '服务'

# 关闭服务
systemctl stop '服务'

# 启动服务
systemctl start '服务'

# 重启服务
systemctl restart '服务'

# 开机自启动服务
systemctl enable '服务'

# 禁用开机自启动服务
systemctl disable '服务'

# 重新加载服务
systemctl reload '服务'

# 查看所有服务
systemctl list-units
```

## 防火墙配置

```sh
# 对外暴露指定端口，协议一般是tcp
firewall-cmd --add-port="端口号/协议" --permanent

# 查看所有开放的端口
firewall-cmd --list-por

# 重新加载防火墙规则
firewall-cmd --reload
```

## 日期和时间

- 格式
  - %Y：表示年
  - %m：表示月
  - %d：表示日
  - %H：表示小时
  - %M：表示分钟
  - %S：表示秒
  - %s：从1970年至今的秒数

```shell
# 查看日期和时间
date

# 查看特定格式的日期和时间
data '+特定格式'

# 日期计算并显示特定格式, 如：date -d '+1 day' +%Y-%m-%d，表示一天后
date -d '运算 单位(时间)' '+特定格式'
```

## 网络服务

### 网络配置

- centos在**/etc/sysconfig/network-scripts/readme-ifcfg-rh.txt**配置

- ubuntu在**/etc/network/interfaces**配置

```shell
# 配置动态ip
auto eth0
iface eth0 inet dhcp

# 配置静态ip
auto eth0
iface eth0 inet static
    address 'ip地址'
    netmask '子网掩码'
    gateway '网关ip'
    
# 配置临时ip
ifconfig "网卡编号" "IP地址"

# 查看网络配置
ifconfig "网卡编号(默认所有网络)"

# 显示指定网卡的ip地址
ip addr show "网卡编号"
```

### 网络基础命令

```sh
# 测试连接
ping 'ip地址/域名'

# 给网络配置IP地址

# 指定检查连接次数
ping -c '连接次数' 'ip地址/域名'

# 查看端口运行状态
telnet "端口号"

# 查看指定ip地址公开暴露的端口,
nmap 'IP地址'

# 发送网络请求
curl 'url地址'

# 发送请求并下载
curl -O 'url地址'

# 查看网卡信息
netstat

# 查看网关信息
route

# 添加网关
route add default gw 192.168.8.2
```

## 状态监控

> 类似于windows1的任务管理器，管理电脑的使用情况

### 系统资源监控

- CPU信息
  - us：用户使用率
  - sy：系统使用率
  - ni：高优先级进程占用百分比
  - id：cpu空闲百分比
  - wa：io率
  - hi：硬件中断率
  - si：软件中断率
  - st：强制等待占比
  - res：物理内存使用情况
- 交互按键：top内容展示的按键
  - **h**：查看帮助画面

```shell
# 查看状态，5秒刷新一次
top

# 只显示特定进程信息
top -p '进程号'

# 指定刷新时长
top -d '时长'

# 显示会创建进程的完整命令
top -c

# 指定刷新次数，刷新多少次后退出
top -n '次数'

# 非交互全屏模式显示
top -b

# 不显示闲置、无用的进程
top -i

# 查找特定用户启动的进程信息
top -u '用户'
```

## 环境变量

- 修改当前用户可用：在**home/.bashrc**文件配置
- 修改所有用户可用：在**etc/profile**文件配置

```shell
# 查看环境变量
env

# 查看指定环境变量的值
echo $KEY

# 设置临时的环境变量
export '变量名'='值'

# 刷新环境变量
source '环境变量文件'

# 添加到可执行环境
export PATH=$PATH:"可执行环境路径"
```
