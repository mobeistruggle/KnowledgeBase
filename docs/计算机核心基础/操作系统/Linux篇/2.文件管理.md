## Linux目录介绍

> 以 / 为根目录

- bin：存放二进制可执行文件(常用命令)
- etc：系统级配置中心，存放所有服务和配置
- home：普通用户的主目录
- usr：存放系统应用程序
- opt：存放应用程序
- proc：虚拟文件系统目录
- root：超级用户、管理员的主目录
- sbin：存放只有root用户才能访问的二进制可执行文件
- dev：存放设备文件
- boot：存放系统引导的使用的文件
- temp：临时文件
- mnt：挂载目录，挂载额外的数据 和 其他操作系统的分区
- srv：存放服务启动时需要使用的数据
- var：存储动态可变数据的目录

## 文件路径

- 绝对路径：从计算机的根(/)开始叫绝对路径
- 相对路径：从当前位置开始叫相对路径
- 特殊路径符
  - **.**：当前目录
  - **..**：上一级目录
  - **~**：home目录

## 特殊符号

- **>**：将左侧命令结果，写入到右侧指定的文件，格式：命令 选项 参数 > 文件路径
- **>>**：将左侧命令结果，追加写入到右侧指定的文件
- **``**：将包含的内容作为命令执行
- **|**：管道符，将左侧的结果作为右侧的输入
- ***:**匹配任意数量字符
- ?:匹配一个字符

|  文件类型  |  值  |        描述        |

## 权限类型

|      权限      | 字符 |               描述               |
| :------------: | :--: | :------------------------------: |
|   读（read）   |  r   |      可以读取文件/目录内容       |
|  写（write）   |  w   |     可以对文件/目录进行修改      |
| 执行（excute） |  x   | 可以执行该文件，对目录使用cd命令 |

### 修改权限

> 也可使用8进制，有x权限记1，r权限记4，w权限记2，如：rwx rwx rwx可以换成777

```shell
# 权限表示 每3个一组
u='用户对文件的权限',g='用户组对文件的权限',o='其他用户对文件的权限'

# 修改指定的文件或目录本身的quan'xian
chmod '权限' '文件/目录路径'

# 修改指定目录及其目录下的文件/目录的权限
chmod -R '权限' '目录路径'
```

## 查看文件信息

> 路径必须是目录， 不写路径默认当前目录, 以 **.** 开头的文件是隐藏文件

```sh
# 查看指定目录下的文件和目录
ls '路径'

# 文件/目录的使用情况
ls -s '路径'

# 查看文件/目录的占用大小（带有单位）
ls -h '路径'

# 以列表形式展现文件/目录信息，首字符表示文件类型, 其余为qu
ls -l '路径'

# 显示所有文件/目录，包含隐藏文件
ls -a '路径'

# 查看文件类型
file "路径"

```

```sh
# 切换到指定路径的目录
cd '路径'

# 直接执行，回到home目录
cd
```

## 创建

```shell
# 创建文件
```

## 移动

> 目标路径不存在，则进行剪切操作

```sh
# 移动文件和目录
mv '源文件路径' '目标目录路径'

# 交互式移动
mv -i

# 强制移动
mv -f

# 显示进度条
mv -v
```

## 复制

```shell
# 复制文件
cp '源路径' '目标路径' 

# 复制源路径的所有子目录或文件
cp -r '源路径' '目标路径'

# 复制目录
cp -a '源路径' '目标路径'

# 交互式复制
cp -i '源路径' '目标路径'

# 显示复制进度条
cp -v '源路径' '目标路径'
```

## 删除

> 参数可以是n个

```shell
# 删除文件
rm '文件路径'

# 递归删除
rm -r '目录路径'

# 强制删除
rm -f '路径'

# 交互式删除
rm -i "路径"

# 删除空目录,目录不为空时报错
rm -d "路径"

# 只能删除空目录,不处理有符号链接的目录
rmdir "路径"
```

## 查看

```shell
# 查看可执行文件所在位置，
which '可执行文件'

# 通过文件名，查看文件位置
find '路径' -name '需要查找的文件名'

# 通过文件大小查看文件位置
find '路径' -size +'数字+单位'		# 查看超过指定大小的文件位置
find '路径' -size -'数字+单位'		# 查看小于指定大小的文件位置

# 文件权限查找
find '路径' -perm '需要查找的文件权限'
```

## 链接文件

> 通过创建一个链接文件，链接到一个文件/目录，会时刻的保持文件的同步性

```shell
# 硬链接：相当于直接把文件/目录复制过来，防止文件误删除
ln '需要链接的文件' '存放链接文件的目录'

# 软链接：只会生成文件/目录镜像，不占用磁盘空间
ln -s '需要链接的文件' '存放链接文件的目录'

```

## 数量统计

> 统计文件的行数，单词数量等，单词数量是以空格为切割的数量

```shell
# 返回指定文件的行号、单词数、字节数
wc '文件路径'

# 统计字节数
wc -c '文件路径'

# 统计字符数
wc -m '文件路径'

# 统计行数
wc -l '文件路径'

# 统计单词数
wc -w '文件路径'
```

## 查看文件内容

```sh
# 查看所有内容,可以查看多个文件
cat '文件路径'

# 分页查看文件内容
more '文件路径'
    
# 查看文件前N行，默认10行
head -n 行数 '文件路径'
    
# 查看文件后N行，默认10行
tail -n "行数" '文件路径' '行数'

# 持续检测指定文件的内容
tail -f '文件路径'

# 根据关键词搜索文件所在行的内容,关键词可以是正则表达式
grep '关键词' '文件路径'

# 显示行号
grep -h '关键词' '文件路径'

# 忽略大小写
grep -i '关键词' '文件路径'

# 取反,输出没匹配到的所有行
grep -v '关键词' '文件路径'
```

## 解压缩

- **tar**：归档文件，将多个文件/目录归档成1个文件，文件体积基本不会压缩太多
- **gzip**：压缩文件，通过算法将多个文件/目录进行大小压缩，文件格式：**.tar.gz、.gz**
- **zip:** 跨平台通用格式

```shell
# 创建压缩文件
tar -c "压缩文件名"

# 列出压缩文件包含的文件
tar "压缩文件路径"

# 显示解压缩进度
tar -v

# 创建文件路径
tar -f '文件路径'

# 解压文件
tar -x

# 以gzip模式使用
tar -z

# 解压到指定路径
tar -C '路径'

# 压缩组合命令，f选项需在最后，要压缩体积是加上-z选项
tar -cvf '压缩文件路径' '需要压缩的文件...'

# 解压命令组合，解压压缩文件时，需要加上-z选项
tar -xvf '压缩文件' -C '解压后的路径'

# 压缩指定路径文件
gzip -9 "路径"

# 解压到指定路径
gzip -d "路径"

# 将文件压缩成zip压缩包
zip '压缩文件路径' '文件' ...

# 将目录压缩成zip压缩包
zip -r '压缩文件路径' '目录' '文件' ...

# 解压到当前目录
unzip '压缩文件'

# 解压到指定路径
unzip '压缩文件' -d '解压后的路径'
```

| 按键     | 描述                       |

| **i**    | 编辑模式，修改文件         |

| o        | 在下一行进入编辑模式       |
| O        | 在上一行进入编辑模式       |
| q        | 在当前行的结尾进入编辑模式 |

| n        | 向下继续搜索               |
| N        | 向上继续搜索               |

| $        | 移动光标到当前行的结尾     |
