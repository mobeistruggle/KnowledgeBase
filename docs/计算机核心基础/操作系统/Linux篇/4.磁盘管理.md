# 磁盘管理

> 管理 磁盘

## df

> 列出文件系统的整体磁盘使用量

```sh
# 格式：df [-ahikHTm] [目录或文件名]

# 显示单位（GB、MB等）
df -h "目录或文件名"

# 显示文件系统类型
df -T "目录或文件名"

# 只显示指定类型的文件系统
df -t "文件系统类型" "目录或文件名"

# 和 -h 选项类型，但不以1024为准，而是以1000为准
df -H "目录或文件名"

# 以KB单位显示
df -k "目录或文件名"

# 显示所有文件系统
df -a "目录或文件名"
```

## fdisk

> 磁盘分区表操作工具

```sh
# 查询指定装置的分区，不带装置 显示 所有
fdisk -l "装置名称"
```
