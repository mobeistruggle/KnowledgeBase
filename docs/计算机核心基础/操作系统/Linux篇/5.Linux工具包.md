## 高级包管理器

> 基于底层管理器，在线安装，类似于应用商店

### yum

> RPM包管理器，用于centos

```shell
# 下载包
yum install '软件名'

# 下载时所有问题默认同意
yum install '软件名' -y

# 查看所有可安装和可更新的包
yum list

# 查看所有已安装的包
yum list installed

# 更新指定包到最新，不填默认
yum update '软件名'

# 删除包
yum remove '软件名'

# 显示rpm包信息
yum info '软件名'

# 清除软件包缓存
yum clean packages

# 下载本地源
yum localinstall '本地' -y

# 清除所有yum源
yum clean all

# 只下载不安装包
yum --downloadonly '软件名'

# 指定下载目录
yum --downloaddir='软件名'

# 指定配置文件
yum -c '配置文件'

# 搜索安装包
yum search '软件名'

# 查看可更新的程序
yum check-update
```

## apt

> DEB包管理器，用于

```shell
# 下载工具
apt install '工具名'

# 删除工具
apt remove '工具名'

# 和yum命令一致
```

## rpm

> 用于安装和管理rpm文件

```shell
# 安装rpm文件
rpm  'rpm文件' -i

# 显示安装过程的信息详细
rpm -v

# 显示安装进度
rom -h

# 卸载软件包
rpm -e

# 查看所有软件包
rpm -a

# 查看软件是否安装
rpm '软件'
```

## 网络工具

## wget

> 通用网络下载器，支持断点续传

```shell
# 下载应用
wget '下载链接'

# 后台下载，将下载信息存放当前目录的wget.log文件
wget '下载连接' -b

# 下载至指定路径
wget '下载链接' -P '目录路径'

# 下载，指定名称
wget '下载链接' -O '名称'

# 断点续传
wget "下载连接" -c
```
