```shell
# 查看输入过命令
history

# 通过命令字符，匹配最近的命令，将其使用
!'命令前缀'		# 最近命令
ctrl+r		   # 久远的命令


clear
```

## 快捷键

|  按键  |       描述       |
| :----: | :--------------: |
|  tab   |     自带补全     |
| ctrl+c |   取消命令执行   |
| ctrl+d |   退出会话管理   |
| ctrl+l |     清空屏幕     |
| ctrl+e |  移动至命令结尾  |
| ctrl+a |  移动至命令开头  |
| 上下键 | 选择输入过的命令 |
|        |                  |
|        |                  |
|        |                  |
