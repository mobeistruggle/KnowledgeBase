## 用户程序处理阶段

1. **编辑阶段:**创建源文件
2. **编译阶段:** 通过编译程序将源文件编译成多个目录模块,生成目标文件
3. **链接阶段:**由链接程序将多个相关目录模块链接在仪器,生成可执行文件
4. **装入阶段:**将程序加载到内存
5. **运行阶段:**程序正式运行

### 绝对装入

> 在编译时,知道程序被分配到的地址,只适用于单道程序环境

- **优点:**时间效率高
- **缺点:**
  - 并发进程数少
  - 编译程序必须知道内存的空闲地址和程序所在内存地址
