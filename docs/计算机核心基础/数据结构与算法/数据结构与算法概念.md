# 初始数据结构与算法

## 数据结构

> 数据结构是组织和存储数据的方式，涵盖数据内容、数据之间关系和数据操作方法

### 特性

- 合理控制空间占用，节省内存
- 操作数据快速
- 提高简洁的数据表示和逻辑信息，便于算法高效运行

## 算法

> 在有限时间内解决特定问题的指令或操作步骤

### 特性

- 问题是明确的，包含清晰的输入和输出的定义
- 具有可行性，能够在有限步骤、时间和内存空间下完成
- 各步骤都有明确含义，在相同输入和运行条件下，输出始终相同

### 复杂度

> 算法性能的重要凭证

#### 时间复杂度

> 算法的运行时间随输入规模增长的变化趋势

- 常数时间：算法的运行时间不随输入而变化，记为O(1)
- 对数时间：随着输入规模增加，运行时间成对数增长，记为O(logn)
- 线性时间：算法的运行时间和输入成正比，记为O(n)
- 线性对数时间：随着输入规模增加，运行时间成线性对数增长，记为O(nlogn)
- 二次时间：运行时间与输入规模的平方成正比，记为O(n²)
- 指数时间：运行时间与输入规模指数级增长，记为O(2的n次方)
- 阶乘时间：运行时间随输入规模阶乘级增长，记为O(n!)

#### 空间复杂度

> 算法运行所需的内存空间随输入规模增长的变化趋势

- 常数空间：算法使用的内存空间不随着输入规模而改变，记为O(1)
- 线性空间：算法使用的内存空间和输入规模成正比，记为O(n)
- 二次空间：算法使用的内存空间与输入规模的平方成正比，记为O(n²)
- 对数空间：随着输入规模的增加，算法的使用空间成对数级增长，记为O(logn)

## 数据结构和算法的关系

- 数据结构是算法的基础，数据结构为算法提供了结构化的存储和操作数据的方法
- 算法为数据结构注入生命力，数据结构和算法结合能解决特定问题
- 算法通常基于不同的数据结构来实现，但执行效率可能相差较大，选择合适的数据结构是关键
