# 算法

> 指在有限的时间内解决特定问题的一组指令或者操作步骤，目标是找到问题以及找出最优解

## 组成要素

- **输入**：算法接收的输入数据或参数。
- .**输出**：算法产生的输出结果或处理后的数据。
- **操作步骤**：算法中任何的具体步骤或操作。
- **控制流程**：算法中的控制流程，用于确定操作步骤的执行顺序，如循环和条件语句。

## 前置概念

- 递归：自上而下解决问题
  - 递：深入调用自身，直到终止条件
  - 归：条件终止后，逐层返回结果
- 迭代：重复一段过程，这个过程基于上一次的结果，自下而上解决问题

## 时间复杂度

> 时间增长趋势

- 常数时间复杂度（O(1)）：算法的执行时间不随输入规模的增加而增加，时间复杂度保持常数级别。
- 线性时间复杂度（O(n)）：算法的执行时间与输入规模成线性关系，随着输入规模的增加而线性增加。
- 对数时间复杂度（O(log n)）：算法的执行时间与输入规模的对数呈线性关系，例如二分查找算法。
- 平方时间复杂度（O(n^2)）：算法的执行时间与输入规模的平方呈线性关系，例如嵌套循环算法。

## 空间复杂度

> 空间增长趋势

- 常数空间复杂度（O(1)）：算法的额外空间需求保持常数级别。
- 线性空间复杂度（O(n)）：算法的额外空间需求与输入规模成线性关系。
