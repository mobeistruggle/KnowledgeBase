## 顺序查找

> 顺序查找是一种按照给定的数据顺序进行逐个处理的算法
>

```python
def sequential_search(list_element, target_value):
    """
    顺序查找算法：也称线性查找算法，从列表初始端到末端进行查找
    :param list_element: 给定的列表
    :param target_value: 要查找的值
    :return:
    """
    inital = 0
    print('要查找的元素', target_value)
    while len(list_element) > inital:
        if list_element[inital] == target_value:
            print('找到了', list_element[inital])
            break
        else:
            inital += 1
        print(list_element[inital])
```

## 二分查找算法

> 二分算法是一种在有序列表中快速查找指定元素的搜索算法

```python
def binary_search(list_element, target_value):
    """
    二分查找算法：也称折半查找算法，列表中项与目标值比较
    :param list_element: 给定的列表
    :param target_value: 需查找的元素
    :return:
    """
    while len(list_element) > 0:
        list_half = len(list_element) // 2
        print('列表中项', list_half)
        if list_element[list_half] > target_value:
            list_element = list_element[:list_half]
            print('左侧一半', list_element)
        elif list_element[list_half] < target_value:
            list_element = list_element[list_half + 1:]
            print('右侧一半', list_element)
        else:
            print('找到了', list_element[list_half])
            break
```

## 广度优先搜索

> **BFS**：按层级查找，从起点开始一直找，若出现分支就先去找其他分支，找完在进行深入查找

## 深度优先搜索

> **DFS**：按分支查找，从起点开始一直查找到尽头，之后再去查找其他分支
