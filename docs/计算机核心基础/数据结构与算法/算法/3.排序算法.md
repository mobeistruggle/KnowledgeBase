## 选择排序

> 临时存储当前值，和后面值做比较，判断是否替换

```python
def select_sort(list_element: list, is_reverse: bool = False) -> list:
    """
    选择排序算法
    :param list_element: 需要排序的列表
    :param is_reverse: bool, 排序方式，默认正序排列
    :return: list 返回排序结果
    """
    for i in range(len(list_element)):
        min_index = i
        for j in range(i + 1, len(list_element)):
            if list_element[min_index] > list_element[j]:
                min_index = j
        if is_reverse:
            list_element[i], list_element[min_index] = list_element[min_index], list_element[i]
        else:
            list_element[min_index], list_element[i] = list_element[i], list_element[min_index]
    return list_element
```
