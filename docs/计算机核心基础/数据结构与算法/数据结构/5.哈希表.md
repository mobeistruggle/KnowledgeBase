# 哈希表

> **Hash：**又称散列表，将键和值做一个映射

- 桶：数组的每一个空位

## 原理

1. 创建一个固定数组(默认长度16)
2. 通过哈希函数算出哈希值
3. 对哈希值和数组长度进行取模运算后存储到数组

## 哈希冲突

> 键(key)冲突

- **原因：**因为是通过取模运算，难免出现计算结果一致
- **解决**
  - **拉链法：**每个桶使用一个链表或红黑树，目标桶存在往里追加
  - **开放寻址法：**冲突时，在运算结果基础上进行其他运算，直至存在空位

## 优劣

- 增删改查速度看情况，最坏情况非常慢，最好情况非常快
- 需要获取所有元素时，非常慢
- 容易出现哈希冲突
