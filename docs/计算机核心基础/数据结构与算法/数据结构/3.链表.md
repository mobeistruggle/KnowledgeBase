# 链表

> **Linked List：**线性结构，存储非连续的数据，由多个节点构成

- 头节点：链表的首个节点，链表的实际数据
- 尾节点：链表的最后一个节点，指向下一个节点的内存地址
- 插入节点：将尾节点指向新的节点，新节点的尾节点指向之前的节点
- 删除节点：将尾节点指向后面第二个节点
- 访问节点：通过尾节点，访问下一个节点的头节点
- 修改节点：找到节点，直接头节点

## 节点

> 链表的最小单位

- 第一个元素：存储上一个节点的内存地址（单链表直接存储数据）
- 其他元素：存储真实数据
- 最后一个元素：存储下一个节点的内存地址

## 单向链表

- 查询速度慢
- 增删数据相对快

## 环形链表

- 头节点指向尾节点

## 双向链表

- 提升了查询速度
- 内存占用大
- 增删数据相对快
