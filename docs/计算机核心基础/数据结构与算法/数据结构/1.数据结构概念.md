# 数据结构

> 用于存储和组织数据的方式，便于不同使用场景

## 逻辑结构

- 线性结构：元素之间一对一关系，如：数组、链表、队列、栈、哈希表
- 非线性结构
  - 树形结构：元素一对多，如：树、堆、哈希表
  - 网状结构：元素多对多，如：图

## 物理结构

- 连续结构：数据之间紧密相连
- 分散结构：数据之间分的很散

## 基本数据类型

> 是CPU可以直接进行运算的类型

- 整型
- 浮点型：以小数表示
- 布尔型：以0和1表示
- 字符型：以单个字符表示

### 数据结构和数据类型的区别

```python
# 数据结构和数据类型是密切相关的概念，但在计算机科学中有着不同的含义。

# 数据结构是一种组织和存储数据的方式，它描述了数据元素之间的关系和操作。它可以包括原始数据类型和组合数据类型。数据结构提供了一种逻辑上的框架，用于表示和处理数据。

# 数据类型是编程语言中的一个概念，用于描述变量或表达式所存储的数据的性质。数据类型定义了数据的取值范围、支持的操作和存储方式。常见的数据类型包括整数、浮点数、字符、布尔值等。编程语言提供了一些内置的数据类型，同时也允许用户自定义数据类型。

# 数据结构和数据类型之间存在紧密关联。具体而言，数据结构通常是由一个或多个数据类型组合而成。例如，线性表数据结构可以使用数组或链表等数据类型来实现。数据结构定义了数据的逻辑组织和操作，而数据类型定义了数据的具体性质和支持的操作。

# 在编程中，我们使用数据类型来创建变量或对象，并使用数据结构来组织和操作这些变量或对象。通过选择适当的数据结构和数据类型，可以提高程序的效率、可读性和可维护性。数据结构和数据类型的选择取决于程序的需求和设计考虑。
```
