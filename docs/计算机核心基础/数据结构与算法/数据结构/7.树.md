# 树

> **tree：**由1以上的有限个节点组成具有层次关系的集合

- **根节点**：位于最顶层，没有父节点

- **节点的度**：一个子节点就是一个度

- **边**：两个节点之间的i

- **树的底**：整个树中，拥有节点的度最多就是树的度

- **叶节点或终端节点**：没有子节点

- **兄弟节点**：具有相同父节点的节点

- **节点的层次**：从根节点开始定义起，根为第一层，根的子节点为第二层，依次类推……

- **树的高度或深度**：整个树中，从根节点到末尾节点的层次

- **堂兄弟节点**：父节点在同一层的节点互为堂兄弟节点

- **节点的祖先**：指一个节点从父节点到根节点所经过的所有节点

- **森林**：多棵互不相交的树的集合称为森林

## 种类

- **无序树：**也称为自由树，树中任意节点的子节点之间没有顺序关系
- **有序树：**树中任意节点的子节点之间有顺序关系
  - **二叉树：**
  - **完全二叉树：**除了最底层的节点，其他节点都满足有两个子节点
    - **满二叉树：**所有节点都有两个子节点
  - **平衡二叉树：**任意节点下的子树的差不大于1的树

## 场景

- XML、HTML等，编写这些文本语言的解析器的时候，不可避免用到树
- 路由协议就是用到了树的算法
- mysql数据库引擎
- 文件系统的目录结构
- 经典的AI算法其实就是树搜索，机器学生中的decision tree也是树结构
