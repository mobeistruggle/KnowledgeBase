# 门电路

> 实现基本逻辑运算和复合逻辑运算的电路单元

## 基础

- 门电路逻辑状态：高、低电平表示1、0
- 开关电路
  - 单开关电路：开关断开输出高电平信号(VCC)、闭合输出低电平信号(0)
  - 互补开关电路：开关电器之间形成互补
- 实现开关的电子元器件
  - 二极管：
  - 三极管：
  - MOS管：栅极、漏极和源极，由栅极控制输入信号，其他为输出信号
    - NMOS管：输入高电平的时候，漏极和源极连通，否则断开
    - PMOS管：与NMOS管相反，栅极输入低电平的时候才连通
    - CMOS：PMOS和NMOS组合

## 基础逻辑电路

### 基础逻辑门

> 通过继电器实现与、或、非门电路

- 与门：and，两者信号为true则为true，否则为False
- 或门：or，两者信号有一个为true则为true，否则为false
- 非门：not，又称反相器，为false的信号则true，为true的信号则false

### 组合逻辑门

- 与非门：NAND，两者信号都为true则为false，否则为true
- 或非门：NOR，两者信号都为false则为true，否则为false
- 异或门：XOR，两者信号不相同为true，否则为false
- 同或门：XNOR，两者信号相同为true，否则为false

## 组合逻辑电路

- 信号计数：计算基础，用来统计信号数量
- 半加器：两个shu'ru信号计数为2就进位
- 全加器：将不同的逻辑门进行组合，构成加法器
- 算术逻辑单元（ALU）：用于不同四则运算的单元，因为加法器是其他四则运算的基础
